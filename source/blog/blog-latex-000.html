Basic Latex packages and templates

Yujie Wang, 2017-11-27

Basic packages

Equations

% in the header
\usepackage{amsmath}
\usepackage{txfonts}

% in the text
\begin{align}
... ...
\end{align}

The amsmath package allows you to list the equations in multiple line, and you may use the align* environment to remove the equation labels. For the one-line equation, you may use the equation environment. More details can be found in a later blog (if any). You may include the txfonts package to sue the times font for the equations.

Framed Text

% in the header
\usepackage{framed}

% in the text
\begin{framed}
... ...
\end{framed}

This helps you outline the important part or independent part.

Geometry

% in the header
\usepackage{geometry}
\geometry{left=2.5cm, right=2.5cm, top=2.5cm, bottom=2.5cm}

This allows you to define the margins. It is easy to customize the margins with above code.

Graphics

% in the header
\usepackage{graphicx}

% in the text
\begin{figure}[htbp]
\includegraphics[width=0.8\textwidth]{figure_name}
\caption{This is the caption.}
\label{figure_label}
\end{figure}

% to ref this figure
Details can be found in \ref{figure_label}.

This command allows you to insert figures in Latex. The figure environment allows graphics to be shown with label and caption. You can use "\ref{}" to make reference to the labels you added.

Hyphenate

% in the header
\usepackage[none]{hyphenat}

If you don't want the hyphenate in the document, you may use the hyphenat package with the option "none".

Listing

% in the header
\usepackage{listing}

% in the text
\lstset{language=C, caption={}, basicstyle=\ttfamily\small}
\begin{lstlisting}
... ...
\end{lstlisting}

The listing environment can be used to paste the programming code into Latex without treating it as Latex file. You may use different settings to set up the environment, such as font size and language type.

Basic templates

Use the following templates for different documents.

Scientific article

\documentclass[12pt, a4paper]{article}

\usepackage{amsmath}
\usepackage{geometry}
\usepackage{graphicx}
\usepackage[none]{hyphenat}
\usepackage{indentfirst}
\usepackage{lineno}
\usepackage[sectionbib]{natbib}
\usepackage{ragged2e}
\usepackage{times}
\usepackage{titlesec}
\usepackage{txfonts} % times font for math
\geometry{top=2.5cm, bottom=2.5cm, left=3.5cm, right=2.5cm}
\linenumbers
\linespread{1.5}
\setlength{\parindent}{2em}
\setlength\RaggedRightParindent{2em}
\titleformat{\section}{\normalfont\fontsize{12}{15}\bfseries}{\thesection}{1em}{}
\titleformat{\subsection}{\normalfont\fontsize{12}{15} \itshape}{\thesection}{1em}{}
% avoid overfull \hbox
\hbadness=10000
\tolerance=10000

% main text
\begin{document}
\RaggedRight
... ...
\end{document}
