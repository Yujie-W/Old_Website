How to use vector graph in Microsoft Office

Yujie Wang, 2019-06-17

Use of SVG figure

In M$ Word, you can insert a SVG file by "Insert -> Pictures -> Choose a SVG Figure". Doing this way, you have the best resolution as long as your SVG file is a real vector graph. This might work for Open Office and Libre Office as well (requires confirmation).
