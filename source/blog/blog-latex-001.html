How to add bibliography to each chapter

Yujie Wang, 2017-11-28

PdfLatex + ChapterBib

PdfLatex is my default Latex compiler. It is easy to include bibliography into the main.tex file and compile it with pdflatex. It would be a little tricky if you need to use individual bibliography in each chapter of your book, such as thesis. To make it easier to organize, you can put each chapter in an individual file, e.g. "./chapter/chapter-1.tex".

In your "main.tex", you should use the package chapterbib by adding this line:

\usepackage[sectionbib]{chapterbib}
\usepackage[sectionbib]{natbib}

The "chapterbib" is the package you want to use, and the "sectionbib" is the option that makes your bibliography a section in the chapter. The "natbib" package ensure that the citation parentheses is the the default (). Then add the following lines in you "chapter-1.tex":

\includebibliography{Your_Bib_File}
\bibliographystyle{apalike}

For the file name of Your_Bib_File, please leave the ".bib" part, e.g. "biblio" but not "biblio.bib". The bibliographystyle can be "apalike", "plain", and etc. Details can be found online.

How to compile

To compile it successfully, you should do as the following make.sh file (command under Linux):

pdflatex main.tex;
sleep 1;
bibtex ./chapter/chapter-1;
sleep 1;
pdflatex main.tex;
sleep 1;
pdflatex main.tex;

The first line is to generate the .aux file for each chapter; the second line is to generate the citation in the given chapter; the last two lines are to generate the final PDF file with right citations. Please not that should can't add ".tex" in the second line because .aux is what bibtex works on. You need to let the file be generated before running the next command, so the "sleep 1" makes the computer sleep for a second for the files to be generated.
