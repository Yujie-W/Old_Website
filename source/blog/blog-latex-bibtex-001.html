How to add bibiography to each chapter

PdfLatex + ChapterBib

PdfLatex is my default Latex compiler. It is easy to include bibiography into the main.tex file and compile it with pdflatex. It would be a little tricky if you need to use individual bibiography in each chapter of your book, such as thesis. To make it easier to organize, you can put each chapter in an individual file, e.g. "./chapter/chapter-1.tex".

In your "main.tex", you should use the package chapterbib by adding this line:

\usepackage[sectionbib]{chapterbib}

The "chapterbib" is the package you want to use, and the "sectionbib" is the option that makes your bibiography a section in the chapter. Then add the following lines in you "chapter-1.tex":

\includebibiography{Your_Bib_File}
\bibliographystyle{apalike}

For the file name of Your_Bib_File, please leave the ".bib" part, e.g. "bibio" but not "bibio.bib". The bibiographystyle can be "apalike", "plain", and etc. Details can be found online.

How to compile

To compile it successfully, you should do as the following (command under Linux):

pdflatex main.tex
bibtex ./chapter/chapter-1
pdflatex main.tex
pdflatex main.tex

The first line is to generate the .aux file for each chapter; the second line is to generate the citation in the given chapter; the last two lines are to generate the final PDF file with right citations. Please not that should can't add ".tex" in the second line because .aux is what bibtex works on.
